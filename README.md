# 짱아통신 사이트

겟몰/엠키즈몰 같은 쇼핑몰형 느낌으로 다시 만든 정적 사이트입니다.

## 수정할 곳
- `script.js`의 `products` 배열: 상품명, 카테고리, 통신사, 가격/조건 수정
- `index.html` 하단 footer: 사업자번호, 통신판매업신고, 대표번호 수정
- `https://pf.kakao.com/` 부분: 실제 카카오톡 채널 주소로 교체

## 실행
압축을 풀고 `index.html`을 더블클릭하면 바로 확인됩니다.

## 운영 추천
현재 버전은 결제/회원가입 없는 상담몰입니다. 개인정보·신분증 업로드 없이 이름/연락처/희망상품만 받아 카카오톡 상담으로 연결하는 구조가 안전합니다.
